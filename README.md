### How to start the HeyDateMe App

Step 1: Start MySQL Workbench server.

Step 2: Open PyCharm.
- Run Main.py (.. python main.py)

Step 3: NPM Start. (node modules required)
- Go to frontend Folder in TERMINAL
- Type npm start in the TERMINAL
